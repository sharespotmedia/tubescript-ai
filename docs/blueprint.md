# **App Name**: TubeScript AI

## Core Features:

- Sidebar Navigation: Implement a sidebar navigation with a red gradient, app branding, content type selector (Vlog, Tutorial, Commentary, Review), feature highlights, and a pro tip callout box.
- Main Content Area: Design a main content area with a topic input field, an optional YouTube/website URL reference field with real-time validation, a generated script display area with copy/download functionality, and a generate button with loading states.
- Style Matching: Implement a tool which analyzes the content creator's style from reference YouTube videos or websites, and applies this style to the generated script.
- Script Generation: Implement a tool to generate video scripts using Claude AI based on the user-provided topic, content type, and optional reference URL, and generate a formatted document available for download.
- Script Templates: Provide pre-built, customizable script templates for common video types and industry-specific formats.

## Style Guidelines:

- Primary color: Saturated red (#F44336), conveying energy and capturing user attention for this video creation platform.
- Background color: Desaturated light red (#F9E7E7) to keep focus on key app functions.
- Accent color: Analogous orange (#FF5722) for interactive elements and generating visual contrast.
- Font pairing: 'Space Grotesk' (sans-serif) for headings and 'Inter' (sans-serif) for body text, providing a contemporary and readable feel.
- Use clean and modern icons, ensuring they are intuitive and relevant to video scripting.
- Implement a responsive layout that adapts to different screen sizes and devices.
